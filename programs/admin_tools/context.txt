Bot Exit:
	End the instance of the bot using "exit()" and file for eternal stop.

Bot Restart:
	Exits the bot but is in a while loop so it restarts.

Cog Control:
	Enable, Disable, Reload any cog with the bot.

Purge:
	Purges messages from a channel, and saves it all to a file in "purge/" named with the time and date of the occurance.
	Puts a message in Audit Log of who sent out the command and stuff (if possible).
